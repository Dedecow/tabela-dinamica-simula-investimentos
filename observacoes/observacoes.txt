Entrega 1

Estrutura do simulador

Objetivos gerais

1. Montar a estrutura base do simulador

2. Integrar as ferramentas JS aprendidas até aqui


Objetivos específicos

1. Declarar variáveis, constantes e arrays - Falta a Array

2. Criar uma ou mais funções JS que gerem interação - ok

3. Adicionar os ciclos de iteração e/ou condicionais necessários, para que seu projeto funcione corretamente - Criar um While, um if e um for 

4. Integra o uso do Console e das caixas de diálogo Prompt, Confirm e Alert - Falta usar o confirm


Deve ser entregue

Documento HTML (pelo menos um) - OK

Arquivo JS referenciado no HTML - OK 

Formato

Arquivo em formato .ZIP com a pasta e os arquivos do projeto. Deve conter o nome “Entregavel1+Sobrenome”

Se você usa mais de um arquivo JS e mais de um arquivo CSS, guarde os mesmos em subpastas

Sugestões

Nesta etapa, concentre-se em realizar a melhor lógica de JS. Não se preocupe com o design web

Não é necessário integrar todos os condicionais nem todos os ciclos de iteração aprendidos.
Apenas aqueles que agreguem valor à lógica de sua aplicação

A interação do simulador será feita integralmente sobre a Consola JS

Descreva no documento HTML com um ou dois parágrafos, a ideia geral do seu simulador - OK

Crie pelo menos 3 funções seguindo o algoritmo básico de todo programa (entradas de dados, processamento de dados e mostrar os resultados de saída dos mesmos.) - FALTA

Faça chamadas (invocar) às funções que você usou em seu simulador - OK

Seu projeto mudará conforme você aprende mais características de JS. O que não conseguir agora, emule da melhor forma possível com variáveis, constantes e/ou arrays

Defina mensagens claras a serem mostradas nas caixas de diálogo. Concatene textos com variáveis, e realize saltos de linha em textos extensos. Isso facilitará sua leitura - OK

Critérios de avaliação:

Estrutura HTML e arquivo JS
A estrutura HTML está completa e executada com boas práticas ou utiliza o starter template do Bootstrap ou outro framework para o uso adequado de HTML5 *. O arquivo JS está corretamente referenciado no HTML.

Algoritmos
Utiliza algoritmo condicional e com ciclo (IF, laços for) de maneira ótima, refletindo o aprendizado em aula.

Funções
Os nomes das funções são claros e indicam que ação realizam. Utiliza a estrutura correta para a montagem das mesmas. Cria funções dinâmicas de maneira correta. Geram um resultado correto quando executadas.

Para te ajudar
Pode-se utilizar o starter template do Bootstrap para agilizar a montagem da estrutura HTML caso desejem. Ou podem optar por aplicar JS aos seus HTML desenvolvidos pelos próprios estudantes (seja em outros cursos ou para este projeto).
